Description

Mongrel application. This module implements the application behaviour. On startup the application creates an ETS table and starts the mongrel supervisor (mongrel_sup).

Function Details

start/2

start(Type::any(), StartArgs::any()) -> {ok, pid()} | {error, Reason::any()}

Starts the application. This function creates an ETS table that is passed to the mongrel supervisor. The table is used to store the field identifiers of records that must be mapped.
